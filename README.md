# digital-forensics-autopsy

## Overview
This repository documents a host-based digital forensics investigation
performed using the Autopsy forensic tool.

The objective was to analyze multiple forensic images and extract
digital evidence related to communication, encrypted data, and
steganographic files.

## Evidence Analyzed
- Windows 11 forensic image
- Android mobile forensic image
- USB drive forensic image

## Tools Used
- Autopsy (DFIR)
- Keyword Search & Timeline Analysis
- Hash Analysis & Metadata Review

## Key Skills Demonstrated
- Digital Forensics & Incident Response (DFIR)
- Evidence Correlation
- Timeline Reconstruction
- Forensic Documentation & Reporting
