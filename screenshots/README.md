# Screenshots

This directory contains a limited set of screenshots captured during the
forensic analysis process.

## Scope of Screenshots
- **Phone Image Analysis** – Evidence extracted from the mobile device
- **USB Image Analysis** – Evidence extracted from the USB storage device
- **Windows Image Analysis** – Evidence extracted from the Windows system

## Privacy and Ethical Notice
To maintain privacy and ethical standards, only a limited number of
**sanitized** screenshots have been included in this repository. Sensitive
information such as personal identifiers, private communications, and
explicit content has been intentionally excluded or obscured.

Additional visual evidence can be demonstrated upon request in a controlled
and appropriate setting.
